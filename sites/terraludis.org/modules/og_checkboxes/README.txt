
-- SUMMARY --

The OG Checkboxes module provides a new widget for the groups' audience fields.
This widget allow to choose audience using checkboxes instead of multiselect 
box.

For a full description of the module, visit the project page:
  http://drupal.org/project/1451296

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/1451296


-- REQUIREMENTS --

* Organic groups http://drupal.org/project/og


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Go to your content type's fields management
* Edit you audience group field
* Go to the widget type tab
* Choose audience groups checkboxes and save


-- TROUBLESHOOTING --

* If the widget does not appear in the list:
  - Empty drupal cache
  - Check if this module is enabled
